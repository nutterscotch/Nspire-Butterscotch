Butterscotch (TI-Nspire)

Butterscotch is an open source re-implementation of GameMaker: Studio's runner.

Generate a TI-Nspire CX II package (ZIP of .tns files) for a GameMaker: Studio game. Tested with Undertale and DELTARUNE.

Upstream Butterscotch Project URL: https://github.com/MrPowerGamerBR/Butterscotch

Tested Hardware

  • TI-Nspire CX II / CX II CAS (primary target, color)
  • MAYBE Nspire CX/CXCAS
Requires Ndless. The runner (engine.tns) is included. Extract the ZIP and copy the bs folder into My Documents on the calculator, then run bs/engine.tns.
